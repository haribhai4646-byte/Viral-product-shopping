⚠ Video generation — connect your own API

Turning these shots into actual moving video clips requires a text/image-to-video API (e.g. Runway, Luma Dream Machine, Pika, or Kling). These services require server-side authentication and billing, so this static page can't embed a working key for you — you'll need to create a free account, grab an API key, and paste it below. Each shot's "Generate Video" button will then call that provider directly from your browser.

Most providers offer free credits to start (typically 5–25 generations). This is the single integration point in the whole app you need to wire up — everything else (script, storyboard, images) already works.
